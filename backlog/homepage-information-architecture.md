# Homepage Information Architecture

## Goal

메인페이지 섹션 순서를 `설명 중심`에서 `행동 중심`으로 바꾼다.

## Scope

- 메인 섹션 순서 재정의
- 각 섹션의 우선순위와 정보량 조정
- 모바일 기준 첫 화면 노출 순서 정리

## Target Structure

1. 히어로 비주얼 + 핵심 소개 + CTA
2. 현재 공개 사건
3. 참여 방법 3단계
4. 종료/발표 대기 사건
5. 운영 지침 / 세계관 설명

## Acceptance Criteria

- 첫 화면에 긴 설명 박스가 오지 않는다.
- 공개 사건 섹션이 메인 상단에 배치된다.
- 세계관 설명은 하단 섹션 또는 접기 UI로 이동한다.
- 모바일에서도 첫 스크롤 전에 사이트 성격과 행동 유도가 동시에 보인다.
