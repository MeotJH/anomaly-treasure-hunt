# Hero Visual And CTA

## Goal

텍스트 중심 상단을 대표 비주얼 중심 히어로로 교체한다.

## Scope

- 히어로 배경 이미지 또는 대표 합성 비주얼 배치
- 짧은 소개 카피 작성
- CTA 2개 배치
- 모바일/데스크톱 높이 조정

## Required Elements

- 작은 배지: 감시 기록 보관
- 큰 제목: 이상현상 추적 기록 보관소
- 짧은 소개 1~2줄
- CTA 예시:
  - 공개 사건 보기
  - 내 제보 보기 또는 제보 시작

## Acceptance Criteria

- 상단에서 텍스트 덩어리 대신 비주얼이 먼저 보인다.
- CTA가 히어로 안에서 바로 보인다.
- 제목 아래 설명은 2줄 안팎으로 제한된다.
- 배경 위 오버레이로 가독성이 확보된다.
