# Archive And Guidance Sections

## Goal

종료된 사건과 긴 설명 콘텐츠를 하단으로 내려 메인페이지 집중도를 높인다.

## Scope

- 과거 조사 문서 섹션 재배치
- 종료/발표 대기 카드 축약
- 운영 지침 / 세계관 설명을 접기 가능한 UI 또는 보조 섹션으로 이동

## Rules

- 과거 사건은 공개 사건보다 시각적 비중이 낮아야 한다.
- 긴 설명은 기본 접힘 또는 짧은 요약 뒤 확장 구조를 우선한다.

## Acceptance Criteria

- 과거 조사 문서는 공개 사건 아래에 배치된다.
- 종료 사건 카드는 작은 썸네일 또는 축약 카드 형태를 갖는다.
- 운영 지침은 기본적으로 메인 주행을 방해하지 않는다.
- 긴 텍스트 블록이 모바일 첫 화면 근처에 남아 있지 않다.
