# Active Case Cards With Thumbnails

## Goal

공개 사건 카드에 사건별 시각 차이를 만들고 참여 유도를 강화한다.

## Scope

- 공개 사건 카드 상단 썸네일 추가
- 카드 요약 문장 2줄 이내로 축약
- 보상 정보와 상태 정보의 위계 재정리
- 공개 중 사건 대표 카드 또는 가로 스와이프 구조 검토

## Card Rules

- 썸네일 비율: 16:9 우선
- 메인 요약: 2줄 이내
- 상태 배지: 공개 중 / 발표 대기 / 종료
- 보상 정보는 별도 강조 블록 유지

## Acceptance Criteria

- 각 사건 카드가 텍스트만으로 구분되지 않는다.
- 썸네일만 봐도 사건 분위기 차이가 느껴진다.
- 카드 한 장에서 제목, 상태, 보상, CTA 우선순위가 분명하다.
- 메인에서 사건 설명이 과도하게 길지 않다.
