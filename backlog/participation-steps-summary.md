# Participation Steps Summary

## Goal

긴 관측 지침 설명을 짧은 참여 방법 3단계로 바꾼다.

## Scope

- 기존 긴 설명 박스 제거 또는 축소
- 3단계 참여 카드 구성
- 각 단계별 짧은 설명 작성

## Target Copy Shape

1. 사건 문서 열람
2. 장소 추리
3. 증거 제출

## Acceptance Criteria

- 메인 중단에 3단계 참여 흐름이 보인다.
- 각 단계 설명은 1~2문장 이내다.
- 사용자가 읽지 않아도 구조만 보고 흐름을 이해할 수 있다.
- 기존 장문의 운영 설명은 이 섹션에 남지 않는다.
