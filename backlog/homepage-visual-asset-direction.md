# Homepage Visual Asset Direction

## Goal

메인페이지와 사건 카드에 들어갈 이미지 자산 방향을 정의한다.

## Scope

- 히어로 대표 이미지 방향 정의
- 사건별 썸네일 컨셉 정의
- 실제 사진 기반 보정 규칙 정리

## Visual Direction

- 실제 장소 사진 기반
- 어두운 보정
- 약한 노이즈와 감시 UI 오버레이
- 파일명 오류, 좌표, 경고선 같은 기록 레이어 허용

## Avoid

- 캐릭터 중심 이미지
- 과도한 판타지풍 일러스트
- 현실 장소성과 무관한 포스터 스타일

## Acceptance Criteria

- 히어로와 사건 카드에 들어갈 자산 종류가 정의된다.
- 사건별 이미지 키워드가 문서화된다.
- 이미지가 사이트 컨셉과 실제 조사 흐름을 동시에 보조한다.
