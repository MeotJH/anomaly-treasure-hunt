# Frontend AGENTS

Apply the operating rules in [Agent.md](Agent.md).

## Focus
- Build the user and admin experience in Next.js App Router.
- Keep routes, components, and state handling aligned to the PRD.

## Non-Negotiables
- No client leakage of answer or code secrets.
- Cover loading, error, expired, and submitted states.
- Prefer server-first data flow and simple state.
